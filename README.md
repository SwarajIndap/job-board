# JobBoard — Full Stack Job Portal

A full-stack job board platform where recruiters can post job openings and job seekers can browse and apply for jobs. Built to practice authentication, REST APIs, relational databases, and modern React UI development.

## Features

- 🔐 **JWT-based authentication** with bcrypt password hashing
- 👥 **Role-based access control** — recruiters and job seekers see different UI/permissions
- 💼 **Job posting** — recruiters can create job listings
- 🔍 **Job listings** — public job feed visible to everyone
- 🎨 **Responsive, modern UI** built with Tailwind CSS
- 🗄️ **Relational database** with Prisma ORM (PostgreSQL)

## Tech Stack

**Frontend:** React, React Router, Axios, Tailwind CSS, Vite
**Backend:** Node.js, Express.js
**Database:** PostgreSQL with Prisma ORM
**Auth:** JWT (JSON Web Tokens), bcrypt
